  • 3D Modelling from 2D images

    May 2017

    This project aimed at creating a program which converted multiple 2D images to a 3D model. The concept used behind the project is that if two images are available, then the position of a 3D point can be found as the intersection of the two projection rays. This process is referred to as triangulation.

    The approach used involved the below processing steps -
    1. Camera calibration consists of intrinsic and extrinsic parameters. The dotted line between calibration and depth determination represents that the camera calibration is usually required for determining depth.
    2. The second phase is Depth determination, it calculates the 3D component missing from any given image depth.
    3. Once you have the multiple depth maps you have to combine them to create a final mesh by calculating depth and projecting.
    4. The final stage is Material Application, which is just material and depth application to the 3D mesh created from the multiple images.
